Aplikasi Augmented Reality Tata Surya

Aplikasi edukasi berbasis Android menggunakan Unity dan Vuforia untuk membantu siswa memahami konsep tata surya.

Capstone Project

Deskripsi Proyek

Proyek ini merupakan aplikasi edukasi berbasis Augmented Reality (AR) berbasis android yang dirancang untuk menghadirkan simulasi tata surya ke dunia nyata. Aplikasi ini dikembangkan menggunakan Unity, Blender, dan Vuforia untuk menciptakan pengalaman belajar. Dengan memanfaatkan teknologi marker-based tracking, pengguna dapat memvisualisasikan model tata surya 3D secara dinamis di lingkungan nyata.

Teknologi yang Digunakan

Unity Vuforia Blender Android
