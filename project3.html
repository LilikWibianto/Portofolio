AR Education

Aplikasi Augmented Reality Sebagai Media Pengenalan Hewan di Mini Zoo Jogja Exotarium

Media edukasi berbasis Android yang membantu pengunjung mengenal berbagai satwa melalui teknologi Augmented Reality.

Kerja Praktik

Oktober 2024 - Januari 2025

Deskripsi Proyek

Proyek ini merancang dan membangun aplikasi Augmented Reality (AR) berbasis Android sebagai media edukasi pengenalan hewan, dimulai dari tahap perencanaan, pembuatan desain antarmuka, hingga implementasi dan pengujian. Proyek ini menggunakan Unity dan Vuforia SDK untuk menampilkan objek 3D beserta deskripsinya, dengan pengkodean dilakukan melalui Visual Studio Code. Selain itu, dilakukan integrasi marker serta pengujian lintas perangkat Android.

Teknologi yang Digunakan

Unity Vuforia Visual Studio Code Blender Canva Android

Hasil Pengembangan

Aplikasi berhasil menampilkan objek hewan berbasis Augmented Reality beserta informasi edukatif yang dapat digunakan oleh pengunjung Mini Zoo Jogja Exotarium. Pengujian menunjukkan aplikasi berjalan dengan baik pada berbagai perangkat Android dan mampu meningkatkan pemahaman pengguna mengenai satwa yang diperkenalkan.
